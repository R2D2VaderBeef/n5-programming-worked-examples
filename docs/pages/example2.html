Example 2

Running Total

Use loops and accumulators to build totals step by step while keeping input valid.

1. Problem Description

Calculate the total cost of 5 items entered by a user.

2. Prediction Activity

How many times will the loop repeat?

3. Step-by-Step Implementation

Step 1: Initialize the total.

total = 0

Step 2: Create a loop for 5 items.

for counter in range(5):

Step 3: Add input to total.

    price = float(input())
    total = total + price

Full Solution: Combine all steps into one complete program.

total = 0
for counter in range(5):
    price = float(input("Price: "))
    total = total + price
print(total)

Drag the lines into the correct logical order:

total = total + price

print(total)
