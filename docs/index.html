Crack the Code helps you move from a problem statement to a finished solution with a repeatable process: predict, build, check, and reflect.

Purpose of the Resource

This resource exists to make problem solving feel manageable and achievable. You will learn how to plan a solution before coding and how to explain what your program is doing.

This resource helps you:

  • Understand how to move from a problem description to working code
  • Learn how loops and algorithms work
  • Practise modifying programs
  • Check your understanding using interactive activities

How to Use the Website

Each example follows the same learning sequence so you always know what to do next.

Each example follows these steps:

  1. Read the problem
  2. Predict the solution
  3. Study the worked example
  4. Complete interactive activities
  5. Modify the program
  6. Reflect on your understanding

Typical time per example: 20 to 30 minutes depending on your pace.

Before You Start

Make sure you are comfortable with these basics:

  • Variables and simple calculations
  • Input and output
  • Using a loop (for or while)

Helpful tips while you work:

  • Write down inputs and outputs before you code
  • Trace your loop with a small example
  • Check each step before moving on

Examples

Work through structured coding activities designed to build confidence step by step. Each example combines prediction, guided implementation, and practice tasks so you can strengthen problem-solving before writing full programs independently.
